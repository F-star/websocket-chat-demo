
WebSocket demo

## 安装

```sh
yarn
```

## 比较原生的实现

```sh
node plain/server.js
```

访问 http://localhost:5050/

## 使用 WebSocket.IO 的实现

```sh
node socket_io/server.js
```

访问 http://localhost:5050/
